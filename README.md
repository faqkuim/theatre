# Theatre IS

Минимальная информационная система «Театр» — Flask + SQLite.

## Быстрый старт

```bash
python -m venv .venv
.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python app.py
```

Откройте http://127.0.0.1:5000

## Сущности БД

| Сущность | Таблица | Описание |
|----------|---------|----------|
| Пользователь | `users` | Данные для входа (email, login, password_hash), ФИО, телефон, флаг `is_admin` |
| Постановка | `productions` | Название спектакля |
| Спектакль | `performances` | Конкретный показ: время, зал, ссылка на постановку |
| Заявка | `requests` | Запрос пользователя на билеты: кол-во, способ оплаты, статус |
| Журнал аутентификации | `auth_log` | Лог входов/регистраций/выходов с IP и User-Agent |

### Статусы заявки

| Код | Название | Описание |
|-----|----------|----------|
| `new` | Новая | Только что создана пользователем |
| `confirmed` | Подтверждена | Администратор одобрил |
| `cancelled` | Отменена | Администратор или система отклонили |

## Сценарии использования

1. **Регистрация** — пользователь заполняет форму (логин, ФИО, телефон, email, пароль).  
   Проверяется уникальность логина и email, минимальная длина пароля (8 симв.).
2. **Авторизация** — вход по email + пароль; при ошибке выводится сообщение.
3. **Личный кабинет** — просмотр предстоящих спектаклей, создание заявки (спектакль + кол-во билетов + способ оплаты).
4. **Список заявок** — пользователь видит свои заявки со статусами.
5. **Административная панель** — администратор видит все заявки, фильтрует по статусу, переводит заявку в «Подтверждена» или «Отменена» (пагинация по 10 штук).

## UI-гайд

### Цвета

| Переменная | Hex | Назначение |
|------------|-----|------------|
| `--bg` | `#0b0d12` | Фон страницы |
| `--card` | `#151923` | Фон карточки |
| `--text` | `#e6ecff` | Основной текст |
| `--accent` | `#6aa0ff` | Акцент, кнопки, ссылки |
| `--bad` | `#ef4444` | Ошибка, отмена |
| `--good` | `#22c55e` | Успех |

### Шрифт

`system-ui, Segoe UI, Roboto, Arial, sans-serif` — системный стек, нет внешних зависимостей.

### Сетка

12-колоночная CSS Grid; на мобильном (≤ 390 px) сжимается до 4 колонок, все элементы — полная ширина.

### Микроанимации

- **Кнопки** — `translateY(-1px)` + `box-shadow` при `:hover`, возврат при `:active`.
- **Карточки** — плавная смена `border-color` при `:hover`.
- **Строки таблицы** — `brightness` на `hover`.
- **Поля ввода** — плавное `border-color` + `box-shadow` при `:focus`.
- **Слайдер** — `opacity` переход 0.8 s; пауза при наведении курсора; навигационные точки.
- **Toast** — `opacity` + `translateY` за 0.25 s.
