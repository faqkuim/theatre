# Студия красоты — IS

Информационная система «Студия красоты» — Flask + SQLite.  
Онлайн-запись, просмотр мастеров и услуг, отзывы клиентов, расписание и панель администратора.

## Быстрый старт

```bash
python -m venv .venv
.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python app.py
```

Откройте http://127.0.0.1:5000

## Сущности БД

| Сущность | Таблица | Описание |
|----------|---------|----------|
| Пользователь | `users` | Данные для входа (login, email, password_hash), ФИО, телефон, флаг `is_admin` |
| Мастер | `masters` | ФИО, специализация, биография, флаг активности |
| Услуга | `services` | Название, описание, цена, длительность, категория |
| Запись | `appointments` | Клиент + мастер + услуга + дата + время + статус |
| Отзыв | `reviews` | Оценка 1–5, текст, ссылки на мастера и услугу, флаг одобрения |
| Журнал входов | `auth_log` | Лог действий (register/login/logout) с IP |

### Статусы записи

| Код | Название | Описание |
|-----|----------|----------|
| `pending` | Ожидает | Только что создана клиентом |
| `confirmed` | Подтверждена | Администратор принял |
| `completed` | Завершена | Визит состоялся |
| `cancelled` | Отменена | Запись отклонена или отменена |

## Сценарии использования

1. **Главная страница** — слайдер, каталог услуг по категориям, карточки мастеров, одобренные отзывы.
2. **Регистрация** — логин, ФИО, телефон, email, пароль (≥ 8 символов); проверка уникальности логина и email.
3. **Авторизация** — вход по email + пароль; при ошибке выводится сообщение.
4. **Личный кабинет** — онлайн-запись (выбор мастера, услуги, даты и времени), список своих записей со статусами, форма отзыва со звёздным рейтингом.
5. **Панель администратора** — фильтрация записей по статусу и мастеру, смена статуса (Подтвердить / Завершить / Отменить), модерация отзывов (Опубликовать / Удалить), пагинация.

## UI-гайд

### Цвета

| Переменная | Hex | Назначение |
|------------|-----|------------|
| `--bg` | `#100d0f` | Фон страницы |
| `--card` | `#1e1520` | Фон карточки |
| `--text` | `#f4e6f0` | Основной текст (тёплый белый) |
| `--accent` | `#d9609a` | Акцент, кнопки, ссылки (розовый) |
| `--bad` | `#ef4444` | Ошибка, отмена |
| `--good` | `#22c55e` | Подтверждение, успех |

### Шрифт

`system-ui, Segoe UI, Roboto, Arial, sans-serif` — системный стек.

### Сетка

12-колоночная CSS Grid; на мобильном (≤ 390 px) сжимается до 4 колонок, все блоки — полная ширина, кнопки тянутся на 100%.

### Микроанимации

- **Кнопки** — `translateY(-1px)` + `box-shadow` при `:hover`.
- **Карточки** — плавное `border-color` при `:hover`.
- **Строки таблицы** — `brightness` на `hover`.
- **Поля ввода** — `border-color` + `box-shadow` при `:focus`.
- **Слайдер** — `opacity` 0.9 s; пауза при наведении; точки-индикаторы.
- **Звёзды отзыва** — CSS hover `~` sibling trick: подсвечивает все звёзды до выбранной.
- **Toast** — `opacity` + `translateY` за 0.25 s.
